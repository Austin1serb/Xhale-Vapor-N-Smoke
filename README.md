# 🌿 HerbaNaturalCo E-Commerce Platform  
> A fully custom full-stack e-commerce solution built for a local CBD business—from storefront to shipping automation.

## 🔥 Project Highlights

- Built with **React**, **JavaScript**, and **Material UI**
- Fully custom **Admin Dashboard** with advanced controls
- Custom **CMS** with full product management interface
- **Order tracking** with automated status-based email updates
- Integrated **Square Payment SDK** for secure checkout
- **Shippo API** for real-time shipping cost calculation & label generation
- **Google Places API** for address autocomplete at checkout
- **Cloudinary** for image storage and optimization
- Achieved **high Lighthouse scores** for both mobile and desktop
- Fully integrated **product CRUD**, **checkout flow**, and **order fulfillment**

---

## 🧱 Tech Stack

- **Frontend**: React, JavaScript, SCSS, Material UI (MUI)
- **Backend**: Node.js, Express.js, MongoDB
- **APIs & Integrations**:
  - **Shippo** – automated shipping + label generation
  - **Google Places** – real-time address autocomplete
  - **Square SDK** – secure payments
  - **Cloudinary** – image hosting and transformation
- **Admin Panel**: Built from scratch with sales reports, inventory management, user roles, and product management
